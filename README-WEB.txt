YG Gacha Adventure - Web v1

部署方式：
1. 將本資料夾內的所有檔案一起上傳至靜態網站空間。
2. 網站入口為 index.html。
3. 不要重新命名 index.html、index.js、index.wasm 或 index.pck。
4. 本版本為單執行緒 WebAssembly，不需要 SharedArrayBuffer 跨來源標頭。
5. 請勿直接用 file:// 雙擊 index.html；本機測試必須透過 HTTP 伺服器。

操作：
方向鍵左右：移動
方向鍵上：跳躍
J：翅膀攻擊
K：蛋殼衝刺
R：失敗或過關後重新開始

第一次進入網頁時，請點擊畫面或按任意鍵開始，以啟用瀏覽器音樂。
